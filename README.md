# API Security Risk Analysis

## Objective
Analyze a public API to identify potential security risks and recommend security improvements.

## Tools Used
- Postman
- JSONPlaceholder API
- Microsoft Word
- GitHub

## API Tested
https://jsonplaceholder.typicode.com/users

## Deliverables
- API Security Risk Analysis Report (PDF)
- API Security Risk Analysis Screenshots (PDF)

## Risks Identified
- No Authentication
- Public Data Exposure
- Authorization Review
- Rate Limiting Review

## Outcome
Successfully analyzed a public API, identified potential security risks, and documented recommendations to improve API security.
